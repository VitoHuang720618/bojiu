# Demo 所见即所得换图功能使用指南

## 功能概述

Demo现在支持所见即所得的图片编辑功能，您可以直接在页面上点击任何图片进行更换，无需使用单独的管理界面。

## 使用方法

### 1. 启用编辑模式
- 在页面右上角点击 **"编辑模式"** 按钮
- 按钮会变成橙色，表示已进入编辑模式
- 轮播图会暂停自动播放

### 2. 更换图片
- 在编辑模式下，所有可编辑的图片会显示虚线边框
- 鼠标悬停在图片上会显示编辑图标和"点击更换图片"提示
- 点击任何图片即可选择新的图片文件
- 支持 JPG、PNG、GIF 等常见图片格式

### 3. 实时预览
- 图片上传后会立即在页面上显示
- 上传进度会显示在图片上方
- 成功上传后会显示"图片更新成功！"消息

### 4. 退出编辑模式
- 点击 **"退出编辑"** 按钮返回正常浏览模式
- 轮播图会恢复自动播放
- 所有编辑提示会隐藏
