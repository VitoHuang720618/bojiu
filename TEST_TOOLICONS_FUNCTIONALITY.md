## 功能特點

### 和背景圖設置一樣的行為
1. **只使用 API 數據**：不使用預設靜態資源作為後備
2. **即時保存**：上傳圖片後立即保存配置並重新載入預覽
3. **動態替換**：API 數據完全替換前端的靜態配置

### 管理介面功能
1. **圖片上傳**：支持默認圖和懸停圖上傳
2. **配置管理**：可設置圖標 ID、名稱
3. **批量操作**：重置為預設配置
4. **即時預覽**：修改後立即在預覽中看到效果

## 測試步驟

1. 訪問管理介面：http://localhost:3002/
2. 切換到 "工具圖標" 標籤頁
3. 上傳自定義圖標圖片
4. 查看預覽頁面確認更新
5. 測試重置功能

## 預期行為

- 工具圖標應該完全由 API 配置控制
- 上傳新圖片後，前端應該立即顯示新圖片
- 沒有 API 數據時，工具圖標區域應該為空（不顯示預設圖標）
- 重置功能應該恢復到預設的 6 個工具圖標配置
